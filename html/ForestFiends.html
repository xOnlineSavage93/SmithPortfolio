Code Review

Below is an example gameplay system used for enemy pursuit. This kind of logic helps give the enemies presence and pressure during combat encounters. 

using UnityEngine;

public class FiendChase : MonoBehaviour
{
    public Transform player;
    public float speed = 3f;
    public float stoppingDistance = 1.5f;

    void Update()
    {
        if (player == null) return;

        float distance = Vector2.Distance(transform.position, player.position);

        if (distance > stoppingDistance)
        {
            transform.position = Vector2.MoveTowards(
                transform.position,
                player.position,
                speed * Time.deltaTime
            );
        }
    }
}

This script demonstrates a simple but effective enemy chase system. It continuously checks the distance to the player and moves the enemy forward until it reaches attack range.
